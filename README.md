🐾 Spa de Mascotas

Aplicación web desarrollada durante horas de clase del curso de programación web. El objetivo del proyecto fue construir una interfaz atractiva y funcional para un spa de mascotas, aplicando conceptos de HTML, CSS y JavaScript.

📖 Descripción

Este proyecto representa una página web para un spa de mascotas donde los usuarios pueden:

Explorar los servicios disponibles
Visualizar productos para el cuidado de sus mascotas
Acceder a un formulario de inicio de sesión

El enfoque principal estuvo en el diseño visual, la estructura de la interfaz y la interacción básica con el usuario.

✨ Funcionalidades
🏠 Landing page
💇 Sección de servicios (peluquería, baños, desparasitaciones)
🛍️ Sección de productos (shampoos, cepillos, pipetas)

🔐 Sistema de autenticación (UI)
Formulario de inicio de sesión (login)
Formulario de registro de usuario
Formulario de recuperación de contraseña
Validaciones básicas en los inputs
👁️ Mostrar / ocultar contraseña

⚠️ Nota: Estas funcionalidades están implementadas a nivel de interfaz (frontend), sin conexión a un backend o base de datos.

🛠️ Tecnologías utilizadas
HTML5
CSS3 (uso de :root para variables)
JavaScript (Vanilla JS)
Git y GitHub
GitHub Pages (para despliegue)
